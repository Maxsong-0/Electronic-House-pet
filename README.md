# Electronic-House-pet
# 电子宠物

#这是一个用 Python 和 PyGame 创建的电子宠物项目。

#功能

#* 虚拟形象：一只可爱的小猫
#* 表情：快乐、悲伤、愤怒、惊讶、饥饿、疲倦等
#* 声音：喵喵叫、呼噜呼噜等
#* 动画：走路、奔跑、跳跃、玩耍等
#* 游戏：抓老鼠、躲猫猫等

### 使用方法

#1. 克隆项目
#2. 安装 Python
#3. 运行 `` 文件
